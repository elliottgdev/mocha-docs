Entity.java

The Entity class is used to make interactive game objects such as the player or an enemy.

declaration

EntityManager takes in a String for the name, a TexturedModel, a Vector3f for the position, three floats for the rotation, and a float for the scale.

Entity entity = new Entity(name, texturedModel, position, rotX, rotY, rotZ, scale);

It is best not to make entities manually. Rather, you should use .ntt files with an EntityManager as it will do this for you.

methods

Entity.getModel();

This method returns the TexturedModel of the Entity.

Entity.setModel(TexturedModel model);

This method changes the model of the Entity to whatever TexturedModel is provided.

Entity.getPosition();

Returns the position of the Entity.

Entity.setPosition(Vector3f position);

Sets the position of the Entity to whatever is provided.

Entity.increasePosition(float dx, float dy, float dz);

Translates the position by whatever is provided.

Entity.setRotX(float rotX);

Sets the X rotation.

Entity.setRotY(float rotY);

Sets the Y rotation.

Entity.setRotZ(float rotZ);

Sets the Z rotation.

Entity.getRotX();

Returns the X rotation.

Entity.getRotY();

Returns the Y rotation.

Entity.getRotZ();

Returns the Z rotation.

Entity.increaseRotation(float dx, float dy, float dz);

Increases the rotation by whatever is provided.

Entity.setScale(float scale);

Sets the scale.

Entity.getScale();

Returns the scale.
