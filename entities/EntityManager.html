EntityManager.java

The Entity Manager class is a useful and easier way of making, storing, and handling entities.

declaration

EntityManager takes in a Renderer, StaticShader, and a Loader.
It takes these in now so you don't have to input these as parameters into EntityManager methods.

EntityManager entityManager = new EntityManager(renderer, shader, loader);

When declared, it will load all the .ntt's in the /src/entities/ntt directory as Entities in the entities ArrayList.

methods

EntityManager.renderEntities();

This method renders all the Entities in the entities ArrayList.
It should be placed in your game's while loop in your game's main script.
example;

while(!Display.isCloseRequested()) {
        renderer.prepare();
        shader.start();
        shader.loadViewMatrix(camera);
        shader.setUniformVector("viewPos", camera.getPosition());
        entityManager.entities.get(entityManager.getEntityByName("test")).increaseRotation(1, 1, 1);
        entityManager.renderEntities();
        shader.stop();
        DisplayManager.updateDisplay();
}


EntityManager.getEntityByName(String name);

This method takes in a String and returns an int.
What this method does is it will compare the names of all the Entities in the entities ArrayList and return the index of the first Entity it finds in the list with a matching name.

Note: It is for this reason that it is adviced to not have multiple entities with the same name

A useful use of this method is to store a reference of an Entity to run code on.
example;

Entity testEntity = entityManager.entities.get(entityManager.getEntityByName("testEntity"));
